# 🥁 Drum Kit

An interactive Drum Kit web application built using **HTML**, **CSS**, and **JavaScript**. Press the corresponding keys on your keyboard or click the drum buttons to play different drum sounds and enjoy a virtual drumming experience.

## 📖 About

This project demonstrates JavaScript event handling, DOM manipulation, and audio playback by creating a fun, interactive drum kit. Each drum button is mapped to a unique sound, and button animations provide visual feedback whenever a drum is played.

## ✨ Features

- 🥁 Play drum sounds by clicking buttons
- ⌨️ Keyboard support for quick playing
- 🎵 Multiple drum sound effects
- ✨ Button animations on key press/click
- 📱 Responsive design for different screen sizes

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

### Clone the repository

```bash
git clone <your-repository-url>
```

### Run the project

1. Open the project folder.
2. Open `index.html` in your preferred web browser.
3. Start playing the drums using your keyboard or mouse!

## 🎮 Controls

| Key | Drum |
|-----|------|
| W | Tom 1 |
| A | Tom 2 |
| S | Tom 3 |
| D | Tom 4 |
| J | Snare |
| K | Crash |
| L | Kick Bass |

> You can also click the drum buttons on the screen to play the sounds.
